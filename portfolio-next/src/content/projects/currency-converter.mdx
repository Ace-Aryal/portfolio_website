## Overview

This simple yet powerful todo app helped me understand how to perform basic CRUD operations in JavaScript. Tasks could be added, marked as done, or deleted, all using vanilla JS.

## Approach

I stored the todos in local storage, allowing persistence across sessions. It gave me a strong grasp of DOM manipulation and event listeners. The UI was built with simple HTML and styled with CSS, keeping everything lightweight.

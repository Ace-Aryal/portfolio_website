## Overview

This simple currency converter app helped me understand how to fetch and work with APIs in JavaScript. It allows users to convert between different currencies in real-time using exchange rate data.

## Approach

I used the `fetch` API to get real-time exchange rates from a public API. The app provided an interactive dropdown to select currencies, with input fields for amounts. It strengthened my understanding of asynchronous JavaScript and user-driven DOM updates. Styling was kept minimal using HTML and CSS to focus on logic.

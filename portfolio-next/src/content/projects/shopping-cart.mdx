## Overview

This Currency Converter App allows users to convert between various global currencies in real-time. It's built with vanilla HTML, CSS, and JavaScript, pulling live rates from a currency API.

## Approach

I designed a clean, responsive UI using Flexbox to ensure a good user experience across devices. JavaScript handles all the logic: fetching exchange rates, populating dropdowns, and updating the converted values as the user inputs amounts or changes currency selections. This project helped me understand asynchronous requests using `fetch()`, DOM updates based on user input, and error handling for real-world APIs.
